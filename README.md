# Chicago Crime Analytics

This project analyzes and visualizes crime patterns in Chicago using the public "Crimes - 2001 to Present" dataset.

Goal
- Explore crime trends over time.
- Visualize the top 10 crime types by year.
- Build interactive charts and maps for storytelling.
 Main Features
- Time-series analysis of crime counts.
- Top 10 crime types by year (including interactive Plotly bar chart).
- Crime heatmap across the city using Folium.
- Static PNG exports for portfolio/GitHub preview.
 Tech Stack
- Python, pandas, numpy
- matplotlib, seaborn
- plotly (interactive charts)
- folium (map visualizations)
- kaleido (export Plotly charts to PNG)
